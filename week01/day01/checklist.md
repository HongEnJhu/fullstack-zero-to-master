## Terminal Drills（在 `week01/day01/lab/`）
> 用「實際檔名」紀錄，不用假路徑（例如 `C:\...\`）。

- [x] Create（建立檔案）
  - Bash：`touch <file>`
  - PowerShell：`New-Item -ItemType File <file>`
- [x] Rename（改名）
  - Bash：`mv <old> <new>`
  - PowerShell：`Rename-Item <old> <new>`
- [x] Copy（複製）
  - Bash：`cp <src> <dst>`
  - PowerShell：`Copy-Item <src> <dst>`
- [ ] Delete（刪除）
  - Bash：`rm <file>`
  - PowerShell：`Remove-Item <file>`
