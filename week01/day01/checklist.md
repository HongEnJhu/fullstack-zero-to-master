### Terminal
在路徑 week01/day01/lab 中
- [x] **Create** ...
    ```terminal
    touch file.ex
    ```
    ```powershell
    New-Item -Path "C:\...\file.ex" -ItemType File
    ```
- [x] **Rename** ...
    ```terminal
    mv Old_Name New_Name
    ```
    ```powershell
    Rename-Item -Path "C:\...\Old_Name" -NewName "New_Name"
    ```
- [x] **Copy** ...
    ```terminal
    cp File1 File2
    ```
    ```powershell
    Copy-Item -Path "C:\...\File1" -Destination "C:\..."
    ```
- [x] **Delete** ...
    ```terminal
    rm File
    ```
    ```powershell
    Remove-Item -Path "C:\...\File"
    ```
